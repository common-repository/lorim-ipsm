== Description ==

This is a simple plugin for generating place holder text and images.

Lorim ipsm block -
Input your options and it will randomly generate a lorim ipsm paragraphs. Pseudo random is used for generating the sentances. Changing the seed will change the sentance. 

Lorim picture block -
Pulls a picture from picsum.photos

== Installation ==
Upload the full directory, with the folder, into your ‘wp-content/plugins’ directory
Activate the plugin through the ‘Plugins’ menu in WordPress
2 new blocks for lorim ipsm and lorim picture will be in your gutenberg
